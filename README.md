# 8QueensInC
Решение задачи о ферзях на поле, N=8. 
Постановка задачи: нужно расставить 8 ферзей на поле так чтобы они не пересекались по горизонтали и вертикали, а также по диагонали. Использован метод рекурсивного перебора.
Ферзи расставляются последовательно, используется проверка "inSafe", при постановке след. ферзя на клетку.
Если клетка считается безопасной, то выполняется поставновка. Если нет, то постановка откатывается назад и ферзь ставится на следующий столбец. 
Задача выполняется до тех пор пока все 8 ферзей не соответствуют условию задачи.
